## 一、行业标准化工作流

影视/短剧解说视频的标准化制作流程为 **6步线性流水线**：

```
素材导入 → 智能分析 → 脚本创作 → 语音合成 → 视频合成 → 导出发布
 Ingest    Analyze    Script     Voice     Compose    Export
```

### 1.1 素材导入（Ingest）

**业务逻辑**：

- 导入原始视频文件（MP4/MOV/AVI/MKV/WebM）
- 提取视频元数据（时长、分辨率、帧率、码率、编码格式）
- 生成缩略图用于预览
- 支持批量导入

**技术需求**：

- FFprobe 视频分析
- 文件 I/O 操作
- 缩略图生成（FFmpeg）

**行业工具**：

- FFprobe（元数据提取）
- yt-dlp / you-get（网络视频下载）

### 1.2 智能分析（Analyze）

**业务逻辑**：

- 场景检测：识别视频中的不同场景（动作/对话/风景/特写等）
- 关键帧提取：提取每个场景的代表性帧
- 音频能量分析：检测音频中的高能量片段（掌声/笑声/音乐/语音）
- 高光片段识别：综合视觉和音频信号识别精彩片段
- 智能分段：将视频按内容逻辑分割

**技术需求**：

- 场景分割算法（基于帧差异）
- 音频分析（能量/过零率/频谱）
- 关键帧提取（FFmpeg）
- 并行处理（场景检测+关键帧+音频分析同时进行）

**行业工具**：

- FFmpeg（视频处理）
- OpenCV（图像处理）
- librosa（音频分析）

### 1.3 脚本创作（Script）

**业务逻辑**：

- 基于视频分析结果，使用LLM生成解说文案
- 支持多种解说风格：
  - 幽默型（humorous）：轻松诙谐
  - 严肃型（serious）：专业深度
  - 对话型（conversational）：亲切自然
  - 悬疑型（suspense）：紧张刺激
  - 温情型（warm）：感人至深
- 文案结构：悬念开头 → 剧情梳理 → 高潮转折 → 结尾点评
- 支持人工编辑和润色

**技术需求**：

- LLM API调用（OpenAI/Claude/DeepSeek/Qwen）
- Prompt Engineering（风格控制、结构引导）
- 文案解析（将长文本分割为时间轴片段）

**行业工具**：

- ChatGPT / Claude（文案生成）
- DeepSeek / 通义千问（中文优化）
- 剪映AI文案（一键生成）

### 1.4 语音合成（Voice）

**业务逻辑**：

- 将解说文案转换为语音
- 支持多种音色选择
- 支持语速、音调、音量调节
- 支持情感控制
- 逐段合成，与脚本时间轴对齐

**技术需求**：

- TTS引擎（Edge TTS / Azure TTS / GPT-SoVITS）
- 音频格式转换
- 时间轴对齐

**行业工具**：

- Edge TTS（免费，微软引擎）
- Azure TTS（高质量，付费）
- GPT-SoVITS（开源语音克隆）
- ChatTTS（开源，情感控制）
- Fish-Speech（开源，多语言）

### 1.5 视频合成（Compose）

**业务逻辑**：

- 按脚本时间轴裁剪原始视频画面
- 生成SRT/ASS格式字幕
- 将解说音频与原始视频音频混合
- 烧录字幕到视频中
- 添加转场效果

**技术需求**：

- FFmpeg滤镜链（裁剪/混合/烧录）
- 字幕格式生成（SRT/ASS）
- 音频混合（amix滤镜）
- 字幕烧录（subtitles滤镜）

**行业工具**：

- FFmpeg（核心视频处理）
- MoviePy（Python封装）
- 剪映（可视化编辑）

### 1.6 导出发布（Export）

**业务逻辑**：

- 多格式导出（MP4/WebM/MOV/MKV/GIF）
- 多分辨率支持（480p/720p/1080p/4K）
- 硬件加速编码（NVENC/VideoToolbox/VAAPI）
- 批量导出

**技术需求**：

- FFmpeg编码（H.264/H.265/VP9/AV1）
- 硬件加速检测
- 进度回调

**行业工具**：

- FFmpeg（编码核心）
- NVENC（NVIDIA硬件编码）
- VideoToolbox（macOS硬件编码）

## 二、行业技术栈总结

### 2.1 核心技术栈

| 环节       | 技术/工具                         | StoryFab对应 |
| ---------- | --------------------------------- | ------------ |
| 视频分析   | FFprobe + FFmpeg                  | ✅ 已实现    |
| 场景检测   | 帧差异算法 + 音频能量             | ✅ 已实现    |
| 关键帧提取 | FFmpeg select滤镜                 | ✅ 已实现    |
| 脚本生成   | LLM API（OpenAI/Claude/DeepSeek） | ✅ 已实现    |
| 语音合成   | Edge TTS / Azure TTS              | ✅ 已实现    |
| 字幕生成   | Whisper / FunASR                  | ✅ 已实现    |
| 视频合成   | FFmpeg 滤镜链                     | ⚠️ 部分实现  |
| 导出       | FFmpeg 编码                       | ✅ 已实现    |

### 2.2 行业趋势（2024-2025）

1. **AI数字人解说**：HeyGen、D-ID、腾讯智影生成真人形象
2. **语音克隆**：GPT-SoVITS（3秒音频克隆音色）
3. **本地化大模型**：DeepSeek、Qwen本地运行降低成本
4. **AI画面生成**：ComfyUI + Stable Diffusion生成封面素材
5. **全自动流水线**：n8n / Dify / FastGPT搭建workflow
6. **AI视频生成**：Sora / Kling / 可灵融入短视频制作

## 五、结论

4. 探索AI数字人和语音克隆能力

- 100%本地处理，隐私安全
