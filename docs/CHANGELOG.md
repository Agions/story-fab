# 更新日志

## v0.3.0 — 2026-07（本期：技术栈迁移 + 重大重构）

### 🎉 新栈迁移

- **桌面框架**: Electron 32 → **Tauri 2**
  - 回归轻量方案，安装包从 ~150MB 降至 ~10MB
  - 内存占用从 ~200MB 降至 ~50MB
  - Rust 后端无缝对接
- **前端框架**: Svelte 5 → **React 18**
  - 回归成熟生态，Vite + Zustand + Tailwind CSS 原栈复用
  - 社区资源丰富，后期维护成本更低
- **原生引擎**: Rust 保留，通过 Tauri Commands 直接调用

### ✨ 新功能

- 剪辑模式：批量多预设导出（最多 5 个预设并行渲染）
- 解说模式：新增 Prompt 模板功能，保存喜欢的风格
- 剪辑模式：搜索转录文本快速定位高光片段
- 脚本编辑：插入停顿 / 转场标记快捷键
- 导出：字幕样式 C（大号描边）一键适配快手 / 强节奏内容

### 🔧 优化

- 导入 2 GB 视频速度提升 30%（Rust 文件操作优化）
- 分析内存占用降低 20%
- 脚本生成 Pipeline 速度提升（策划与撰写并行）
- 编辑器 UI 响应速度提升

### 🐛 修复

- 修复 Windows 中文路径导入失败
- 修复大型项目切换页面时内存泄漏
- 修复 TTS 试听时偶发的卡顿
- 修复导出失败时残留临时文件未被清理
