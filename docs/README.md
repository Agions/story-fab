## 核心能力概览

- `AI 解说`：基于画面理解生成结构化解说文本
- `AI 混剪`：根据节奏和场景自动剪辑并组织片段
- `第一人称视角解说`：按 POV 叙事风格组织文案
- `自动添加原画`：关键流程支持原画叠加与辅助镜头
