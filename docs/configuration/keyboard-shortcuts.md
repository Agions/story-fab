# 键盘快捷键

StoryFab 支持完整的键盘操作，脱离鼠标也能高效完成工作。

## 全局快捷键

| 功能 | macOS | Windows / Linux |
| --- | --- | --- |
| 偏好设置 | `⌘` + `,` | `Ctrl` + `,` |
| 新建项目 | `⌘` + `N` | `Ctrl` + `N` |
| 打开项目 | `⌘` + `O` | `Ctrl` + `O` |
| 保存 | `⌘` + `S` | `Ctrl` + `S` |
| 搜索 | `⌘` + `F` | `Ctrl` + `F` |
| 撤销 | `⌘` + `Z` | `Ctrl` + `Z` |
| 重做 | `⌘` + `⇧` + `Z` | `Ctrl` + `Shift` + `Z` |
| 关闭窗口 | `⌘` + `W` | `Ctrl` + `W` |
| 退出应用 | `⌘` + `Q` | `Ctrl` + `Q` |
| 全屏 | `⌃` + `⌘` + `F` | `F11` |
| 切换侧边栏 | `⌘` + `B` | `Ctrl` + `B` |

## 播放器控制

| 功能 | 快捷键 |
| --- | --- |
| 播放 / 暂停 | `Space` |
| 逐帧后退 | `J` |
| 逐帧前进 | `L` |
| 后退 5 秒 | `←` |
| 前进 5 秒 | `→` |
| 后退 1 秒（逐帧非精确） | `Shift` + `←` |
| 前进 1 秒（逐帧非精确） | `Shift` + `→` |
| 跳到开头 | `Home` |
| 跳到结尾 | `End` |
| 音量加 | `↑` |
| 音量减 | `↓` |
| 静音 | `M` |

## 时间轴操作

| 功能 | 快捷键 |
| --- | --- |
| 打入点 | `I` |
| 打出点 | `O` |
| 清除入出点 | `Option` + `I` / `Alt` + `I` |
| 手动插入高光标记 | `M` |
| 拆分片段（在入出点处） | `S` |
| 合并选中片段 | `G` |
| 删除选中片段 | `Delete` / `Backspace` |
| 跳上一高光标记 | `Shift` + `←` 或 `[` |
| 跳下一高光标记 | `Shift` + `→` 或 `]` |
| 缩放时间轴（更密） | `⌘` + `+` / `Ctrl` + `+` |
| 缩放时间轴（更宽） | `⌘` + `-` / `Ctrl` + `-` |
| 缩放时间轴（重置） | `⌘` + `0` / `Ctrl` + `0` |

## 脚本编辑

| 功能 | 快捷键 |
| --- | --- |
| 编辑模式切换 | `E` |
| 插入停顿（0.5s） | `P` |
| 插入转场标记 | `T` |
| 复制段落 | `⌘` + `C` / `Ctrl` + `C` |
| 粘贴段落 | `⌘` + `V` / `Ctrl` + `V` |
| 删除段落 | `Delete` / `Backspace` |
| 上移段落 | `⌘` + `↑` / `Ctrl` + `↑` |
| 下移段落 | `⌘` + `↓` / `Ctrl` + `↓` |
| 试听当前段落 | `T` + `Enter` |

## 导出操作

| 功能 | 快捷键 |
| --- | --- |
| 打开导出面板 | `⌘` + `E` / `Ctrl` + `E` |
| 开始导出 | `Enter`（面板内） |
| 取消导出 | `Esc` |

## 视图切换

| 功能 | 快捷键 |
| --- | --- |
| 切到项目页 | `1` |
| 切到剪辑模式 | `2` |
| 切到解说模式 | `3` |
| 切到导出页 | `4` |
| 切到设置页 | `5` |

## 提示

1. 快捷键在输入框内输入时会被禁用，切换到非焦点状态即可恢复。
2. 时间轴和输入框内的快捷键不同，注意焦点位置。
3. 可在「设置 → 快捷键」中自定义。
