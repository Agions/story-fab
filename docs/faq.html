常见问题

关于 StoryForge 的常见问题解答。

如何开始使用 StoryForge？

请参考快速开始指南，了解如何安装和启动项目。

StoryForge 支持哪些视频格式？

StoryForge 支持 MP4、MOV、AVI、MKV 等常见视频格式。

是否需要 AI API 密钥？

是的，需要配置 AI 模型服务才能使用 AI 分析和生成功能。请参考 AI 模型配置 指南。

StoryForge 是免费的吗？

StoryForge 本身是开源免费的，但使用 AI 服务（如 OpenAI、Azure 等）可能需要付费。

如何获取 AI API 密钥？

你可以在 OpenAI、Azure、百度智能云等平台申请 API 密钥。详细配置请参考 AI 模型配置。

导出时支持哪些格式？

支持剪映草稿、Premiere Pro、Final Cut Pro、DaVinci Resolve 项目文件，以及 MP4、MOV 等标准视频格式。
