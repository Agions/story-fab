# 功能特性

> StoryForge 提供完整的 AI 视频创作解决方案

---

## 🎬 核心剪辑模式

### 🔥 智能混剪

AI 自动识别视频中的高光时刻（掌声、笑点、动作高潮），一键生成精彩集锦。

| 能力 | 说明 |
|------|------|
| 场景检测 | 自动识别镜头边界 |
| 音频峰值 | 检测笑声、掌声、音乐高潮 |
| 运动分析 | 识别动作强度变化 |
| 批量处理 | 支持多视频同时分析 |

**适用场景：** 活动回顾、体育高光、综艺片段、直播精华

---

### 🎭 剧情分析模式 ✨

理解视频叙事结构，分析剧情节点，生成故事驱动的智能剪辑。

| 能力 | 说明 |
|------|------|
| 剧情图谱 | 自动标注每个剧情节点的时间戳 |
| 情感追踪 | 识别画面情感变化（喜悦/悲伤/紧张/平静）|
| 节点类型 | 分辨 Setup / Rising / Climax / Falling / Resolution |
| 多版本输出 | 剧情完整版 / 精华版 / 高能混剪版 |

**适用场景：** 纪录片剪辑、访谈整理、活动视频、剧情类内容

---

### 🎤 AI 解说模式

分析视频内容，自动生成专业解说词，并合成配音。

| 能力 | 说明 |
|------|------|
| 多模型支持 | GPT / Claude / Gemini / 通义千问 等 |
| 解说模板 | 提供多种风格预设 |
| 语音合成 | 支持多种音色和语速 |
| 多语言 | 中文 / 英文 / 日文 等 |

**适用场景：** 知识类视频、产品介绍、纪录片旁白

---

### 📝 第一人称视角模式

将第三人称素材转化为具有个人叙事感的视角体验。

| 能力 | 说明 |
|------|------|
| 视角重构 | AI 自动调整叙事角度 |
| 人称转换 | 第三人称 → 第一人称 |
| 风格适配 | 保留原有画面，重新配音 |

**适用场景：** Vlog、旅拍、个人故事类内容

---

### 🔄 视频重混模式

智能重组视频，保留叙事逻辑同时确保内容原创性。

| 能力 | 说明 |
|------|------|
| 智能选取 | AI 选择符合叙事逻辑的片段 |
| 原创检测 | 避免重复片段 |
| 自动排序 | 按叙事逻辑重新排列 |

**适用场景：** 二次创作、素材重组、内容聚合

## 📦 支持的格式

### 输入格式

| 格式 | 说明 |
|------|------|
| MP4 | 通用格式，推荐使用 |
| MOV | QuickTime 格式（macOS）|
| WebM | Web 优化格式 |
| AVI / MKV | 传统格式 |

### 输出格式

| 格式 | 说明 | 适用场景 |
|------|------|---------|
| MP4 | 兼容性最好 | 全平台通用 |
| WebM | 文件更小 | 网页嵌入 |
| MOV | 高质量 | Mac 编辑 |
| MKV | 灵活封装 | 专业制作 |

### 输出分辨率

| 分辨率 | 适用场景 |
|--------|---------|
| 720p | 快速预览、社交媒体 |
| 1080p | **推荐**，通用场景 |
| 4K | 高质量输出 |
