# 功能介绍

CutDeck 提供完整的 AI 视频创作解决方案，涵盖从素材导入到成片导出的全流程。

---

## 核心功能

### 🎬 AI 智能拆条（v1.7.0 新增）

**长视频 → 多短片段**，一键分发全平台：

| 能力 | 说明 |
|------|------|
| 6 维评分 | 笑声密度 / 情感峰值 / 完整度 / 静默比 / 节奏 / 关键词 |
| SEO 元数据 | 自动生成标题 / 描述 / Hashtags |
| 多格式导出 | 9:16 竖屏（抖音） / 1:1 方屏（小红书） / 16:9 横屏（B站） |
| 平台适配 | 抖音 / 小红书 / B站 / YouTube Shorts / TikTok |

详见：[AI 智能拆条指南](./guide/clip-repurpose.md)

---

### 🎬 剧情分析

AI 深度理解视频叙事结构，自动识别：

| 能力 | 说明 |
|------|------|
| 高光时刻 | 自动识别视频中最精彩的片段 |
| 情感曲线 | 分析视频的情感起伏变化 |
| 关键帧 | 提取最具代表性的画面 |
| 场景分割 | 智能识别不同场景的边界 |
| 叙事结构 | 理解视频的起承转合 |

**适用场景**：纪录片剪辑、演讲视频、赛事集锦

---

### ✂️ 智能剪辑

基于 AI 分析结果，提供两种剪辑模式：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **自动剪辑** | AI 全自动生成剪辑方案 | 快速产出 |
| **半自动剪辑** | AI 提供建议，人工决策 | 质量把控 |

**剪辑参数**：

| 参数 | 选项 |
|------|------|
| 目标时长 | 固定时长 / 范围时长 / 跟随内容 |
| 节奏感 | 慢 / 中 / 快 |
| 转场效果 | 无 / 基础 / 创意 |
| 高光密度 | 低 / 中 / 高 |

---

### 📐 多格式导出（v1.7.0 新增）

Rust 层 FFmpeg scale+crop filter，实时裁切：

| 格式 | 比例 | 尺寸 | 推荐场景 |
|------|------|------|----------|
| **竖屏** | 9:16 | 1080×1920 | 抖音 / 快手 / TikTok |
| **方屏** | 1:1 | 1080×1080 | 小红书 / Instagram |
| **横屏** | 16:9 | 1920×1080 | B站 / YouTube |

---

### 🎙️ AI 配音

多音色 AI 配音，支持丰富的参数调节：

| 功能 | 说明 |
|------|------|
| 多音色选择 | 男声/女声，多种风格 |
| 语速调节 | 0.5x - 2.0x |
| 音调调节 | -50% ~ +50% |
| 多语言 | 中文、英文、日文、韩文等 |
| 情感调节 | 平静 / 活力 / 温柔 |

**推荐音色**：

| 音色 | 适用内容 |
|------|----------|
| 男声-磁性 | 知识讲解、故事叙述 |
| 女声-温柔 | 生活分享、情感内容 |
| 男声-活力 | 科技评测、游戏 |
| 女声-甜美 | 美食、旅行、时尚 |

---

### 🎵 智能混剪

自动识别节奏卡点，多素材智能拼接：

| 能力 | 说明 |
|------|------|
| 节拍检测 | 自动分析音频节奏 |
| 智能拼接 | 多素材自动合成 |
| 转场优化 | 创意转场效果 |
| 风格统一 | 色彩、节奏风格一致 |

---

### 📝 字幕生成

支持多种字幕生成方式：

| 方式 | 说明 |
|------|------|
| **AI 生成**（新增） | Whisper ASR 驱动，秒→毫秒精度时间轴对齐 |
| ASR 自动生成 | 从音频自动识别文字 |
| OCR 识别 | 从画面中提取文字 |
| 导入字幕 | SRT / ASS / VTT 文件 |
| 手动输入 | 自由编辑字幕 |

**字幕样式**：

| 样式 | 特点 |
|------|------|
| 电影字幕 | 黑底白字，居中显示 |
| 弹出字幕 | 半透明背景，跟随说话人 |
| 动感字幕 | 打字机、缩放动画效果 |
| 简约白字 | 无背景白色文字 |

---

### 🎨 自动包装

智能化的视频包装功能：

| 功能 | 说明 |
|------|------|
| 字幕样式 | 多种预设模板 + 自定义 |
| 封面设计 | AI 生成专业封面 |
| 片头片尾 | 自动生成片头片尾 |
| 品牌水印 | 添加专属水印 |

---

## 导出格式

CutDeck 支持多种导出格式：

| 格式 | 说明 | 推荐场景 |
|------|------|----------|
| **MP4** | 通用格式，画质高 | 各平台直接发布 |
| **MOV** | Apple 生态 | ProRes 二次编辑 |
| **剪映草稿** | 导入剪映 | 精细调色、特效 |
| **PR 项目** | 导入 Premiere | 专业后期 |
| **DaVinci 项目** | 导入 DaVinci | 专业调色 |
| **FCP 项目** | 导入 Final Cut Pro | Apple 专业生态 |

### 平台优化预设

| 平台 | 推荐设置 |
|------|----------|
| 抖音/快手 | 9:16 竖屏，1080P，H.265 |
| B站 | 16:9 横屏，1080P/4K，H.264 |
| YouTube | 16:9 横屏，4K，H.264 |
| 小红书 | 3:4 或 9:16，1080P |

---

## 工作流程

完整的视频创作流程：

```
1. 导入素材 → 支持多种视频格式
2. AI 分析 → 智能识别内容结构
3. 生成文案 → AI 生成解说词
4. 视频合成 → 智能拼接与转场
5. 字幕包装 → 多样式字幕
6. 导出成片 → 多格式导出
```
