# 功能介绍

CutDeck 提供完整的 AI 视频创作解决方案，涵盖从素材导入到成片导出的全流程。

---

## 核心功能

### AI 智能剪辑

**长视频 → 多短片段**，一键分发全平台：

| 能力 | 说明 |
|------|------|
| 6 维评分 | 笑声密度 / 情感峰值 / 完整度 / 静默比 / 节奏 / 关键词 |
| SEO 元数据 | 自动生成标题 / 描述 / Hashtags |
| 多格式导出 | 9:16 竖屏（抖音） / 1:1 方屏（小红书） / 16:9 横屏（B站） |
| 平台适配 | 抖音 / 小红书 / B站 / YouTube Shorts / TikTok |

详见：[AI 智能拆条指南](./guide/clip-repurpose.md)

---

### 剧情分析与高光检测

Rust 层真实分析调用：

| 能力 | 说明 |
|------|------|
| 场景分割 | `detect_smart_segments` 智能分段 |
| 关键帧提取 | VisionService 真实提取 |
| 高光检测 | `highlight_detector.rs` FFmpeg scdet + 音频能量分析 ✅ 已激活 |
| 情感分析 | VisionService 多维情感识别 |

---

### 智能剪辑

基于 AI 分析结果，提供两种剪辑模式：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **自动剪辑** | AI 全自动生成剪辑方案 | 快速产出 |
| **半自动剪辑** | AI 提供建议，人工决策 | 质量把控 |

**剪辑参数**：

| 参数 | 选项 |
|------|------|
| 目标时长 | 固定时长 / 范围时长 / 跟随内容 |
| 节奏感 | 慢 / 中 / 快 |
| 转场效果 | 无 / 基础 / 创意 |
| 高光密度 | 低 / 中 / 高 |

---

### 多格式导出

Rust 层 FFmpeg scale+crop filter，实时裁切：

| 格式 | 比例 | 尺寸 | 推荐场景 |
|------|------|------|----------|
| **竖屏** | 9:16 | 1080×1920 | 抖音 / 快手 / TikTok |
| **方屏** | 1:1 | 1080×1080 | 小红书 / Instagram |
| **横屏** | 16:9 | 1920×1080 | B站 / YouTube |

---

### 语音合成

浏览器原生 Web Speech API 驱动，无需外部服务：

| 功能 | 说明 |
|------|------|
| 多音色 | 男声 / 女声 / 中性 |
| 语速调节 | 0.1x - 2.0x |
| 音调调节 | 0 - 2 倍 |
| 多语言 | 中文（zh-CN）优先 |

::: tip 说明
当前使用浏览器内置语音引擎。如需更自然的合成效果，可接入 ElevenLabs 或 Azure TTS。
:::

---

### 自动配乐

内置预设音乐库 + AI 匹配推荐：

| 能力 | 说明 |
|------|------|
| 预置音乐 | 多种风格/情绪分类 |
| 本地上传 | 支持自定义音乐文件 |
| AI 匹配 | 根据视频内容推荐音乐 |
| 节拍对齐 | 自动对齐音频节拍点 |

::: tip 说明
当前版本预置音乐库已可用，AI 推荐功能持续完善中。
:::

---

### 字幕生成

支持多种字幕生成方式：

| 方式 | 说明 |
|------|------|
| **Whisper ASR**（v1.7.0）| faster-whisper 本地转写，秒级精度时间轴 |
| OCR 识别 | Tesseract.js 浏览器端识别 |
| 导入字幕 | SRT / ASS / VTT 文件 |
| 手动输入 | 自由编辑字幕 |

**字幕样式**：

| 样式 | 特点 |
|------|------|
| 电影字幕 | 黑底白字，居中显示 |
| 弹出字幕 | 半透明背景，跟随说话人 |
| 动感字幕 | 打字机、缩放动画效果 |
| 简约白字 | 无背景白色文字 |

| 格式 | 说明 | 推荐场景 |

| 平台 | 推荐设置 |
|------|----------|
| 抖音/快手 | 9:16 竖屏，1080P，H.265 |

| YouTube | 16:9 横屏，4K，H.264 |
| 小红书 | 3:4 或 9:16，1080P |

## 工作流程

完整的视频创作流程：

```
导入素材 → AI 分析 → AI 拆条 → 字幕生成 → 智能剪辑 → 多格式导出
  1          2          3           4           5           6
```

| 步骤 | 输入 | 处理 | 输出 |
|------|------|------|------|
| 1. 导入素材 | MP4 / MOV / AVI / MKV / WebM | 本地解析 | 时间轴 clip |
| 2. AI 分析 | 视频流 | 场景分割 + 高光检测 + 情感分析 | AI 分析结果 |
| 3. AI 拆条 | AI 分析 | 6 维评分 + SEO 元数据 | 多个短片段候选 |
| 4. 字幕生成 | 视频流 | Whisper ASR 本地转写 | SRT 字幕文件 |
| 5. 智能剪辑 | 片段候选 | 自动/半自动剪辑方案 | 完整时间线 |
| 6. 多格式导出 | 时间线 | FFmpeg 裁切 + 编码 | 9:16 / 1:1 / 16:9 MP4 |
