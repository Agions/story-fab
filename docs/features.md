# 功能介绍

StoryForge 提供完整的 AI 视频创作功能。

## 🎬 剧情分析

智能分析视频的叙事结构、情感曲线和高光时刻。

**功能特点：**
- 📊 叙事结构分析（起承转合）
- 🎭 情感曲线绘制
- ✂️ 智能剪辑点建议
- 🌟 高光时刻提取

## 🎙️ AI 视频解说

将视频自动转化为带配音的解说版本。

**功能特点：**
- 🎤 多音色 AI 配音
- 📝 自动生成解说词
- 🔄 音画自动同步
- 🎨 多种解说风格

## 🎵 智能混剪

多素材智能拼接，自动卡点。

**功能特点：**
- 🎵 BPM 节拍检测
- 🔀 智能转场匹配
- ⚡ 并行处理
- 📋 批量导出

## 🎭 情感独白

画面情感分析，生成电影级独白。

**功能特点：**
- 😊 情感识别
- 💭 AI 文案生成
- 🎬 电影感字幕
- 🎭 多种情感风格

## 🤖 AI 模型支持

| 类型 | 支持的模型 |
|------|-----------|
| 文本生成 | GPT-4o, Claude 4, Gemini, Qwen, GLM-4, Kimi, DeepSeek R1 |
| 视觉理解 | GPT-4o, Claude 4, Gemini, Qwen-VL |
| 语音合成 | Edge TTS (免费), OpenAI TTS |

## 📤 导出格式

| 格式 | 说明 |
|------|------|
| 剪映草稿 | `.draft.json` |
| Adobe Premiere | `.xml` |
| Final Cut Pro | `.fcpxml` |
| DaVinci Resolve | `.drp` |
| 视频文件 | `.mp4`, `.webm` |
| 字幕文件 | `.srt`, `.ass`, `.vtt` |

## 工作流

StoryForge 支持完整的工作流：

1. **项目创建** → 选择剪辑模式
2. **素材导入** → 视频上传和预处理
3. **AI 分析** → 剧情分析/解说生成
4. **剪辑合成** → 智能剪辑和合成
5. **导出发布** → 多格式导出

详见 [工作流](workflow) 文档。
