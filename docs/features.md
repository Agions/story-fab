# 功能介绍

StoryForge 提供完整的 AI 视频创作解决方案，涵盖从素材导入到成片导出的全流程。

## 核心功能

### 剧情分析

AI 深度理解视频叙事结构，自动识别高光时刻与情感曲线，提取关键帧和场景信息。

### 智能剪辑

基于 AI 分析结果，一键生成专业级剪辑方案，支持多种剪辑策略和模板。

### AI 配音

多音色 AI 配音，支持中文、英文及多种方言，可调节语速、音调等参数。

### 智能混剪

自动识别节奏卡点，多素材智能拼接与转场，生成节奏感强的混剪视频。

### 字幕生成

支持 ASR 自动字幕提取和 OCR 字幕识别，多格式字幕文件导出。

### 自动包装

智能字幕样式、封面设计，片头片尾生成，让视频更加专业。

## 导出格式

StoryForge 支持多种导出格式，适配不同平台需求：

| 格式 | 说明 |
|------|------|
| 剪映草稿 | 兼容剪映项目文件 |
| Premiere Pro | PR 项目文件 (.prproj) |
| Final Cut Pro | FCP 项目文件 |
| DaVinci Resolve | DaVinci 项目文件 |
| 标准视频 | MP4、MOV 等通用格式 |

## 工作流程

完整的视频创作流程：

1. **导入素材** - 支持多种视频格式导入
2. **AI 分析** - 智能分析视频内容
3. **生成文案** - AI 生成解说词
4. **视频合成** - 智能拼接与转场
5. **导出成片** - 多格式导出
