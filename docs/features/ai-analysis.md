# 解说模式

解说模式围绕 Agent Pipeline 自动 **生成解说脚本 + TTS 配音 + 字幕烧录**，适合知识解说、短剧搬运、影视综述等场景。

## 工作流（5 步）

```
选择片段 → 5步Pipeline生成脚本 → 编辑润色 → TTS配音 → 渲染导出
```

## 第一次使用

### 1. 选择目标片段

时间轴上选择一个已有高光片段，或手动设置入出点。

视频长度建议：
- **短视频**（≤60s）：直接使用原始片段
- **中等长度**（1-5分钟）：AI 会自动识别重点片段，再展开解说
- **长片段**（>5分钟）：建议先转场分段处理

### 2. 启动分析

点击「生成解说」后自动进入 **5 步 Agent Pipeline**：

| 步骤 | 内容 | 时间参考 | 可干预 |
| --- | --- | --- | --- |
| 1. 解析 | 理解视频内容，提取画面语义和已有字幕 | 5-15s | — |
| 2. 策划 | Director Agent 规划叙事节奏，约 3-7 个段落 | 10-30s | 主动告知 AI 你的偏好 |
| 3. 撰写 | 基于策划生成解说词初稿 | 10-30s | 风格/语气调整 |
| 4. 审核 | 检查时长、内容连贯性、敏感词 | 2-5s | 手动调整 |
| 5. 润色 | 优化停顿、语气词、衔接词 | 5-15s | 一键应用 |

每步结果右侧实时预览，可随时中断修改。

### 3. 编辑脚本

脚本编辑支持：

- **段落级编辑**：点击任意段落直接修改文本
- **插入停顿**：选中位置按 `P` 插入 0.5s 停顿
- **调整语速**：段落右侧滑块调整 0.5x-2.0x
- **插入转场**：段落间按 `T` 插入转场特效标记

### 4. TTS 配音

#### 选择引擎

| 引擎 | 音色数量 | 是否联网 | 音质 |
| --- | --- | --- | --- |
| Edge TTS | 40＋ | 是（仅首次） | 中等 |
| Azure TTS | 100＋ | 是（仅首次） | 高 |

#### 试听流程

1. 选择一个音色（默认男声 / 女声各 1 个）
2. 点击「试听」听到完整配音
3. 满意后点击「确认」

#### 批量试听

支持分段试听：
- 选中一个段落单独试听
- 选中多个段落批量试听
- 不满意点击「重新生成」重新获取

### 5. 渲染导出

渲染包含：

1. **视频处理**：硬字幕烧录（可选）+ 转场效果 + 比例调整
2. **音频处理**：TTS 音频与原声混音（可选降低原声）
3. **编码输出**：H.264 / H.265 + AAC，支持自定义码率

## 进阶操作

### 自定义 Prompt

在「高级」选项卡中可以注入额外的指令：

```
"这是一个纪录片风格的解说，避免使用网络用语，语速中等偏慢，
重点突出历史背景的交代"
```

Prompt 会覆盖默认的 Director Agent 策略。

### 多版本对比

1. 点击「保存当前版本」
2. 调整后再次点击「保存当前版本」
3. 最多保存 5 个版本
4. 切换对比打分

### 配音替换

TTS 生成后，也可以随时：
- 导入自己的录音文件
- 选择不同的 TTS 引擎重新生成
- 手动调整音频轨道音量

## 推荐参数组合

| 场景 | 比例 | 码率 | 字幕样式 | 音色推荐 |
| --- | --- | --- | --- | --- |
| 短视频解说 | 9:16 | 12 Mbps | 居中、大号、描边 | 男声沉稳 |
| 教育内容 | 16:9 | 16 Mbps | 底部、中号、半透明底框 | 女声清晰 |
| 影视综述 | 16:9 | 16 Mbps | 底部、小号、白色 | 男声磁性 |
