# 剪辑模式

剪辑模式专注于 **把长视频快速切成高光片段**，适合直播回放、游戏集锦、会议记录等场景。

## 工作流（3 步）

```
导入视频 → AI 识别高光 + 手动微调 → 多比例批量导出
```

## 第一次使用

### 1. 导入视频

直接拖入或点击上传区域，支持 `.mp4` `.mov` `.avi` `.mkv` `.webm`。

### 2. 启动分析

点击「开始分析」→ 并行启动：

- **Whisper 转录**：音频转文字，生成时间戳
- **高光检测**：基于音频能量 + 画面变化 + 语音活动三维度识别

进度条实时显示，支持后台运行。

### 3. 查看与调整

分析完成后，时间轴上自动标记高光片段：

| 操作 | 方法 |
| --- | --- |
| 预览片段 | 点击片段或按 空格 |
| 调整入点 | 拖动片段左边缘 / 按 I |
| 调整出点 | 拖动片段右边缘 / 按 O |
| 拆分片段 | 在目标位置按 S |
| 合并片段 | 选中相邻片段按 M |
| 删除片段 | 选中后按 Delete |
| 调整灵敏度 | 配置面板滑动条（默认中等） |

### 4. 批量导出

一次性导出所有高光片段：

1. 点击「批量导出」
2. 选择比例（9:16 / 1:1 / 16:9）
3. 选择分辨率和码率
4. 勾选「硬字幕烧录」（可选）
5. 点击「开始」

## 进阶操作

### 手动标记高光

自动检测遗漏时，手动标记精彩瞬间：

- 播放过程中按 `M` 在播放头位置打标记
- 后续自动合并相邻标记为片段
- 结合 J / K / L 逐帧精确定位

### 搜索转录文本

搜索关键词快速定位片段：

1. 按 `Cmd/Ctrl + F` 打开搜索框
2. 输入关键词
3. 搜索结果高亮显示在时间轴上
4. 点击跳转可直接定位

### 多版本导出

对同一视频导出多个版本（比例不同、片段不同）：

1. 在「导出预设」中保存当前配置
2. 切换预设即可一键切换导出参数
3. 最多支持 5 个预设

## 快捷键速查

| 功能 | 快捷键 |
| --- | --- |
| 播放 / 暂停 | 空格 |

## 推荐工作流

```
直播录像（2 小时）
  → 启动分析（约 5 分钟）
  → 调整检测灵敏度（多保留 20%）
  → 浏览自动片段，删除不合适的
  → 手动补充遗漏的高光点
  → 选择 9:16 + 硬字幕预设
  → 批量导出 10+ 条短片
  → 直接分发抖音/快手/视频号
```
