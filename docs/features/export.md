# 导出与分发

StoryFab 支持多比例、多编码器输出，并直接适配各平台分发要求。

## 导出参数

### 分辨率与比例

| 比例 | 分辨率 | 适用平台 |
| --- | --- | --- |
| 9:16 | 1080×1920 | TikTok / YouTube Shorts / 快手 |
| 1:1 | 1080×1080 | Instagram Feed / 小红书 |
| 16:9 | 1920×1080 | YouTube / B站 / 腾讯视频 |
| 4:5 | 1080×1350 | Instagram Portrait |
| 21:9 | 2560×1080 | 超宽屏演示 |

### 编码器

| 编码器 | 兼容性 | 压缩率 | 速度 |
| --- | --- | --- | --- |
| H.264 (libx264) | 最高（所有平台） | 中等 | 快 |
| H.265 (libx265) | 高（除老设备） | 高（同画质体积-30%） | 慢 |
| ProRes | 仅专业软件 | 极低（文件巨大） | 快 |

**推荐**：YouTube / TikTok / 抖音用 H.264；本地存档用 H.265。

### 码率控制

| 分辨率 | 推荐码率 | 说明 |
| --- | --- | --- |
| 1080p（9:16） | 12 Mbps | 短视频平台推荐 |
| 1080p（16:9） | 16 Mbps | YouTube / B站推荐 |
| 4K | 40-60 Mbps | 高质量存档 |

### 帧率

- **30 fps**：通用，适合解说、Vlog
- **60 fps**：运动、游戏、高流畅度场景

## 字幕样式

字幕烧录（硬字幕）支持三种样式：

### 样式 A — 底部标准（默认）

```
底边距：5%
字号：屏幕高度的 4%
颜色：#FFFFFF
背景：底部 3% 半透明黑条
```

适合：YouTube、B站、教育类

### 样式 B — 居中醒目

```
垂直居中偏下 15%
字号：屏幕高度的 5%
颜色：#FFFFFF
描边：黑色 2px
```

适合：TikTok、抖音、竖屏短视频

### 样式 C — 大号描边

```
底部 8%
字号：屏幕高度的 6%
颜色：#FFFFFF
描边：黑色 3px
阴影：黑色 软阴影
```

适合：快手、强节奏内容

### 自定义样式

| 参数 | 范围 | 说明 |
| --- | --- | --- |
| 字体 | 系统可用字体 | 跨平台需确保目标设备有同款 |
| 字号 | 2%-10% 屏幕高度 | 竖屏推荐 5-6% |
| 颜色 | 十六进制颜色码 | 默认白色 |
| 描边 | 0-5px | 增大可提升可读性 |
| 底边距 | 0%-20% | 竖屏建议 8-10% |

## 音频处理

### TTS 混音

导出时 TTS 音频与原声混合：

| 参数 | 范围 | 推荐 |
| --- | --- | --- |
| TTS 音量 | 0-100% | 80-90% |
| 原声保留 | 0-50% | 10-20%（背景氛围） |
| 淡入淡出 | 0-3 秒 | 0.5 秒 |

### 音频编码

| 编码器 | 码率 | 说明 |
| --- | --- | --- |
| AAC | 128-320 kbps | 通用 |
| MP3 | 128-320 kbps | 兼容老设备 |

## 多平台分发预设

内置一键预设：

| 预设 | 比例 | 分辨率 | 码率 | 字幕样式 |
| --- | --- | --- | --- | --- |
| TikTok | 9:16 | 1080×1920 | 12 Mbps | B 居中醒目 |
| YouTube Shorts | 9:16 | 1080×1920 | 12 Mbps | B 居中醒目 |
| YouTube 标准 | 16:9 | 1920×1080 | 16 Mbps | A 底部标准 |
| B站 | 16:9 | 1920×1080 | 16 Mbps | A 底部标准 |
| 小红书 | 1:1 | 1080×1080 | 10 Mbps | A 底部标准 |

## 批量导出

支持一次性导出同一视频的多个版本：

1. 点击「批量导出」
2. 勾选多个预设
3. 点击「开始」
4. 所有版本并行渲染

## 剪辑模式批量切条

剪辑模式下，对每个高光片段：

1. 勾选「单独导出每个片段」
2. 选择导出预设
3. 点击「批量导出」

所有片段按顺序渲染并保存到指定目录。
