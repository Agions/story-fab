# 脚本生成

基于视频内容自动生成解说脚本。

## 工作流

```
视频 + Whisper 字幕
        │
        ▼
  上下文组装（视频元数据 + 字幕摘要 + 风格偏好）
        │
        ▼
  LLM 生成（10 家 Provider 可选）
        │
        ▼
  脚本解析（分段 + 时间戳对齐）
        │
        ▼
  质量检查（一致性 / 长度 / 风格）
        │
        ▼
  输出（JSON + Markdown 双格式）
```

## Prompt 模板

每种风格有独立模板：幽默、震惊、感动、专业、纪录片。

模板位置：`src/core/services/providers/prompts.ts`。

## 风格控制

| 风格 | 长度 | 句式 |
| --- | --- | --- |
| 幽默 | 中等 | 短句 + 反转 |
| 震惊 | 短 | 感叹 + 数字 |
| 感动 | 长 | 描述 + 抒情 |
| 专业 | 中等 | 术语 + 解释 |
| 纪录片 | 长 | 叙述 + 引用 |

## 缓存与成本

- 同视频 + 同风格 + 同 Provider = 命中本地缓存，不重复调用 LLM
- 缓存位置：`<config-dir>/script-cache/<video-hash>.json`
- LLM 调用成本：每分钟视频约 0.05-0.20 美元（视 Provider 而定）

## 重生成

不满意可一键重新生成，会用相同输入但不同随机种子。可手动编辑后保存为模板。
