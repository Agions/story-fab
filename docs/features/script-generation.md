# 脚本生成原理

StoryFab 的脚本生成基于 **5 步 Agent Pipeline**，由 Director Agent 全流程把控叙事节奏和内容质量。

## Pipeline 架构

```
视频素材
  → [1. 解析] 画面语义 + 字幕文本 + 音频情绪
  → [2. 策划] 段落切分 + 每段核心信息点 + 节奏曲线
  → [3. 撰写] 基于策划生成初稿（口语化、带停顿标记）
  → [4. 审核] 时长校验 + 敏感词过滤 + 逻辑一致性
  → [5. 润色] 语气词优化 + 衔接词调整 + 最终定稿
```

## 各步骤详解

### 1. 解析（Extract）

**输入**：视频帧（每秒 1 帧）+ 字幕 Whisper 转录 + 音频能量

**输出**：结构化素材库

- 画面关键内容（人物、场景、动作）
- 字幕全文 + 时间戳
- 音频情绪曲线（平静 / 高潮 / 低谷）

**耗时**：约 5-15 秒（本地推理，不调 API）

### 2. 策划（Plan）

**Director Agent 核心决策**：

| 决策项 | 逻辑 |
| --- | --- |
| 段落数量 | 每 30-60 秒视频切为一个段落 |
| 信息密度 | 高潮段落多写、过渡段落少写 |
| 节奏控制 | 开头快速切入 → 中段展开 → 结尾升华 |
| 重点标注 | 标记每段必须保留的信息点 |

**耗时**：10-30 秒

### 3. 撰写（Write）

基于策划结果生成初稿：

- **口语化**：避免书面语，模拟真实解说风格
- **停顿标记**：自动插入 `‖` 标记表示 0.5s 停顿
- **时长控制**：按每分钟 150 字估算，适配目标时长

### 4. 审核（Review）

- **时长校验**：确保配音时长 ≤ 视频时长
- **敏感词过滤**：内置 1000+ 敏感词库
- **逻辑一致性**：检查段落间是否有矛盾

### 5. 润色（Polish）

- 语气词优化（"那么"、"其实"、"你会发现"）
- 段落衔接词调整
- 朗读流畅度评分

## Director Agent 风格

可在「高级」中选择：

| 风格 | 说明 | 适合场景 |
| --- | --- | --- |
| 纪录片 | 沉稳、客观、节奏偏慢 | 科普、人文 |
| 综艺 | 活泼、调侃、节奏快 | 娱乐、搞笑 |
| 教学 | 清晰、结构化、重点突出 | 教育、技术 |
| 影评 | 客观分析、适当主观评价 | 影视解说 |
| Vlog | 亲切、像朋友聊天 | 生活、日常 |

## 自定义 Prompt

支持注入额外指令覆盖默认行为：

```
"这是一个纪录片风格的解说，避免使用网络用语，
语速中等偏慢，重点突出历史背景，
不要强调人物情感，客观陈述为主。"
```

### Prompt 编写技巧

- ✅ **指定风格**：纪录片 / 综艺 / 教学 / 影评
- ✅ **指定语速**：偏慢 / 中等 / 偏快
- ✅ **指定重点**：突出 XX、弱化 YY、不要提 ZZ
- ✅ **指定语气**：客观、亲切、严肃、幽默
- ❌ **不要指定具体文本**：Prompt 只控制风格不控制具体内容

### 保存为模板

满意的 Prompt 可保存为模板，下次直接选用：

1. 输入 Prompt
2. 点击「保存为模板」
3. 命名（如"纪录片风格"、"游戏解说"）
4. 下次在模板列表中直接点击选用

## 多轮改写

点击「重新生成」后：
- 保留 Prompt
- 重新执行全流程
- 最多生成 5 个版本并排对比
