快速开始

本指南将帮助你在 5 分钟内启动 StoryForge 项目。

环境要求

要求版本
Node.js≥ 18.0.0
npm≥ 9.0.0
Git≥ 2.30.0

安装步骤

克隆项目

从 GitHub 克隆 StoryForge 仓库到本地

git clone https://github.com/Agions/StoryForge.git
cd StoryForge

安装依赖

使用 npm 安装项目依赖

npm install

启动开发服务器

运行开发服务器

npm run dev

开始创作

打开浏览器访问 http://localhost:1430

提示

首次启动可能需要几分钟时间来安装依赖和构建项目。
