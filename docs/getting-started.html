快速开始 ​

本指南将帮助你在 5 分钟内启动 StoryForge 项目。

环境要求 ​

要求版本说明
Node.js≥ 18.0.0推荐 LTS 版本
npm≥ 9.0.0随 Node.js 一起安装
Git≥ 2.30.0用于版本控制

安装步骤 ​

1. 克隆项目 ​

bash
git clone https://github.com/Agions/StoryForge.git
cd StoryForge
1
2

2. 安装依赖 ​

bash
npm install
1

3. 配置环境变量 ​

复制环境变量模板并配置：

bash
cp .env.example .env
1

编辑 .env 文件，配置你的 AI API 密钥。详细配置请参考 AI 模型配置。

4. 启动开发服务器 ​

bash
npm run dev
1

打开浏览器访问 http://localhost:1430

提示

首次启动可能需要几分钟时间来安装依赖和构建项目。
