# 产品介绍

StoryFab 是一款 **本地优先** 的 AI 影视解说创作桌面应用。它能把长视频自动转成带有解说配音、字幕、高光的短视频片段，全部在你的电脑上完成，原始素材 **100% 不上传任何云端**。

## 两种工作模式

| 模式 | 适合谁 | 核心流程 |
| --- | --- | --- |
| **剪辑模式** | 直播主、游戏玩家、Vlog 创作者 | 导入视频 → AI 自动识别高光 → 切分片段 → 多比例导出 |
| **解说模式** | 知识博主、短剧解说、内容运营 | 导入视频 → Whisper 转录 → AI 生成解说词 → TTS 配音 → 字幕烧录 → 成片导出 |

## 核心能力

- **本地 Whisper 转录**
  使用 faster-whisper 在本地完成音频转文字，支持中英双语。首次启动会自动下载约 150MB 的模型文件，之后完全离线可用。

- **AI 高光检测**
  基于音频能量峰值、画面变化率、语音活动三维度自动识别精彩片段，无需人工拖动时间轴。

- **5 步 Agent Pipeline**
  解析 → 策划 → 撰写 → 审核 → 润色，由 Director Agent 全流程把控叙事节奏和时长。

- **双 TTS 引擎**
  - **Edge TTS**：完全免费，支持 40+ 中文音色，无需 API Key
  - **Azure TTS**：音质更自然，需自行申请 API Key

- **多比例硬字幕导出**
  输出 9:16 / 1:1 / 16:9 / 4:5 / 21:9 任意比例，字幕直接烧录进视频，适配 TikTok、Instagram、YouTube 各平台。

## 技术亮点

| 层级 | 技术 | 特点 |
| --- | --- | --- |
| 桌面壳 | Tauri 2 | 安装包 ~10MB，内存 ~50MB，Rust 后端无缝对接 |
| 前端 UI | React 18 | 组件化、生态成熟、开发效率高 |
| 原生引擎 | Rust | 高性能 FFmpeg 调用 + 文件系统操作，通过 Tauri Commands 直接调用 |

## 适用场景

- 📹 **短视频批量切条**：把 1 小时直播回放切成 10+ 条高光片段
- 🎬 **影视解说创作**：电影 / 短剧 / 综艺的 AI 解说 + TTS 配音
- 📚 **教育内容分发**：课堂录像加字幕、加解说，多平台分发
- 🎙️ **播客可视化**：音频转录 + 高光画面 + 字幕自动组合
