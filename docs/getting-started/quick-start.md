# 快速开始

5 步从素材到成片。

## 1. 新建项目

打开 StoryFab，点击「新建项目」，命名并选择输出目录。

## 2. 导入视频

支持 `.mp4`、`.mov`、`.avi`、`.mkv`、`.webm`，单文件最大 2GB。拖入或点击上传。

## 3. AI 分析

并行运行两个进程：

- **Whisper 转录**：完整音频转字幕，存本地
- **高光检测**：基于音频能量、画面变化、语音活动识别精彩片段

分析耗时约为视频时长的 1-2 倍，进度实时显示。

## 4. 选择模式

**剪辑模式**：在时间轴上查看高光片段 → 拖拽调边界 → 预览 → 导出。

**解说模式**：选择片段 → 生成解说词 → 编辑脚本 → TTS 试听 → 渲染。

## 5. 导出

| 平台 | 比例 | 分辨率 |
| --- | --- | --- |
| TikTok / Shorts | 9:16 | 1080×1920 |
| Instagram Reels | 9:16 | 1080×1920 |
| Instagram Feed | 1:1 | 1080×1080 |
| YouTube | 16:9 | 1920×1080 |
| Twitter / X | 16:9 | 1920×1080 |

开启「烧录字幕」可把字幕直接嵌入视频。

## 键盘快捷键

| 快捷键 | 功能 |
| --- | --- |
| 空格 | 播放 / 暂停 |
| I / O | 设置入点 / 出点 |
| J / K / L | 逐帧（后退 / 暂停 / 前进） |
| Cmd/Ctrl + Z | 撤销 |
