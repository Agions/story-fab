# 批量处理指南

StoryForge 的批量处理功能让你能够 **同时处理多个视频**，大幅提升创作效率。

---

## 什么是批量处理？

批量处理是指一次操作处理多个视频素材，自动套用相同的 AI 分析和剪辑策略。适合以下场景：

- 🎬 **系列视频** — 同一主题的多个片段一键剪辑
- 📺 **账号矩阵** — 多个账号同时产出内容
- 🏢 **企业内容** — 产品视频、宣传片的规模化生产
- 📚 **教程制作** — 批量处理课程录像

---

## 开启批量处理

### 入口

主界面左侧导航栏 → **「批量处理」** 面板

### 添加素材

| 方式 | 操作 |
|------|------|
| 拖拽添加 | 直接将视频文件/文件夹拖入处理列表 |
| 批量选择 | `Ctrl + 点击` 多选，或 `Shift + 点击` 区间选择 |
| 文件夹导入 | 点击 **「导入文件夹」**，自动扫描并添加所有视频 |

::: tip 提示
批量处理支持 **文件夹递归扫描**，子文件夹中的视频也会被自动添加。
:::

## 批量处理模式

### 模式一：统一策略

所有视频使用 **相同的 AI 分析和剪辑策略**。

```
设置流程：
1. 选择「统一策略」模式
2. 配置 AI 分析选项（剧情分析 / 内容摘要 / 情感分析）
3. 设置剪辑参数（时长、节奏、转场）
4. 点击「开始批量处理」
```

**适用场景**：系列视频、风格统一的批量剪辑

### 模式二：智能适配

AI 根据每个视频的 **内容特点自动选择最佳策略**。

```
设置流程：
1. 选择「智能适配」模式
2. 设置时长范围和基本要求
3. 点击「开始批量处理」
4. AI 自动分析每个视频并生成剪辑
```

**适用场景**：内容多样的素材混剪

## 批量处理配置

### AI 分析配置

| 选项 | 说明 |
|------|------|
| 分析深度 | 快速 / 标准 / 深度（影响处理时间和质量） |
| 分析内容 | 高光检测 / 情感分析 / 场景分割 / 文字识别 |
| 输出文案 | 是否生成配套解说文案 |

### 剪辑配置

| 选项 | 说明 |
|------|------|
| 输出时长 | 固定时长 / 范围时长 / 跟随内容 |
| 目标时长 | 设置具体的输出时长（如 60 秒、3 分钟） |
| 转场效果 | 无 / 基础 / 创意（渐变、缩放等） |
| 字幕样式 | 跟随原始 / 统一模板 / AI 匹配 |

### 导出配置

| 选项 | 说明 |
|------|------|
| 输出格式 | MP4 / MOV / 剪映草稿 / PR 项目 |
| 输出画质 | 原始 / 1080P / 4K |
| 命名规则 | 原始名 / 序号 / 自定义前缀 |

## 队列管理

批量处理采用 **队列机制**，可以实时监控处理进度。

### 队列面板

```
┌─────────────────────────────────────────────┐
│  批量处理队列                        [暂停]  │
├─────────────────────────────────────────────┤
│  ⏳ 视频_001.mp4    ████████░░  80%   AI分析  │
│  ⏸ 视频_002.mp4    ░░░░░░░░░   0%    等待中  │
│  Ⓛ 视频_003.mp4    ██████░░░░  60%   剪辑中  │
│  ⏸ 视频_004.mp4    ░░░░░░░░░   0%    等待中  │
├─────────────────────────────────────────────┤
│  进度：1/4 完成   预计剩余：12 分钟           │
└─────────────────────────────────────────────┘
```

### 队列操作

| 操作 | 说明 |
|------|------|
| 暂停全部 | 暂停整个队列 |
| 暂停单项 | 暂停单个视频的处理 |
| 跳过 | 跳过当前视频，继续下一个 |
| 优先级 | 调整单个视频的处理优先级 |
| 移除 | 从队列中移除（不影响原文件） |

## 模板与预设

StoryForge 支持将批量处理配置保存为 **模板**，方便复用。

### 创建模板

1. 配置好批量处理参数后
2. 点击 **「保存为模板」**
3. 输入模板名称和描述
4. 选择是否设为默认模板

### 使用模板

1. 在批量处理面板选择 **「加载模板」**
2. 选择已有模板
3. 点击 **「应用」** 即可一键配置

### 内置模板

| 模板名 | 适用场景 |
|--------|----------|
| 短视频切片 | 批量将长视频切为 15-60 秒短视频 |
| 高光集锦 | 从多个视频中提取高光片段 |
| 统一风格 | 批量套用相同剪辑风格 |
| 快速预览 | 快速生成低码率预览版本 |

## 批量处理与 AI 配额

::: warning 注意
批量处理会消耗 AI API 配额，每个视频的分析和剪辑都会调用 AI 服务。
:::

| 分析模式 | 预估 API 消耗 |
|----------|--------------|
| 快速分析 | ~500 tokens / 视频 |
| 标准分析 | ~2000 tokens / 视频 |
| 深度分析 | ~5000 tokens / 视频 |

::: tip 建议
- 大量处理时建议使用 **DeepSeek** 或 **Qwen**，性价比最高
- 可以开启 **批量处理的智能限流**，避免触发 API 速率限制
:::

## 常见问题

### 批量处理中断了怎么办？

StoryForge 支持断点续传。重新打开批量处理面板，之前处理失败的任务会显示在「未完成」列表中，点击 **「继续处理」** 即可。

### 处理过程中可以添加新任务吗？

可以。队列运行时，新添加的视频会自动排在队列末尾，不会影响正在处理的任务。

### 输出文件被覆盖了？

批量处理默认启用 **防重复命名**，同一次批量处理中不会产生文件名冲突。如果输出目录已有同名文件，系统会自动添加序号后缀。
