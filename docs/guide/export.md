# 导出格式详解

CutDeck 支持多种导出格式，适配不同的发布平台和后期编辑需求。

---

## 导出格式一览

| 格式 | 类型 | 适用场景 |
|------|------|----------|
| MP4 | 视频文件 | 各大平台直接发布 |
| MOV | 视频文件 | Apple 生态、专业后期 |
| 剪映草稿 | 项目文件 | 导入剪映精细编辑 |
| PR 项目 | 项目文件 | 导入 Premiere Pro |
| DaVinci 项目 | 项目文件 | 导入 DaVinci Resolve |
| FCP 项目 | 项目文件 | 导入 Final Cut Pro |
| MKV | 视频文件 | 多轨封装、存档 |

---

## 视频文件格式

### MP4（H.264 / H.265）

最通用的视频格式，兼容性最佳。

| 设置项 | 选项 |
|--------|------|
| 编码 | H.264 / H.265 |
| 分辨率 | 原始 / 4K / 1080P / 720P |
| 码率控制 | CRF / ABR / VBR |
| 码率值 | 5-50 Mbps |
| 帧率 | 原始 / 60 / 30 / 24 |
| 音频编码 | AAC / MP3 |
| 音频码率 | 128 / 192 / 256 / 320 kbps |

::: tip 推荐设置
- **抖音/快手**：H.265, 1080P, 8-15 Mbps
- **B站**：H.264, 1080P/4K, 10-20 Mbps
- **YouTube 4K**：H.265, 4K, 35-50 Mbps
:::

### MOV

Apple QuickTime 格式，适合 Final Cut Pro 和 macOS 生态。

| 设置项 | 特点 |
|--------|------|
| 编码 | ProRes 422 / ProRes 4444 / H.264 |
| 色彩空间 | Rec.709 / Rec.2020 / DCI-P3 |
| 色彩深度 | 8-bit / 10-bit / 12-bit |
| Alpha 通道 | 支持（ProRes 4444） |

::: tip 推荐场景
- 进一步用专业软件调色
- 需要 Alpha 通道（透明背景）
- 存档用途（ProRes 质量最高）
:::

### MKV

开源封装格式，支持多音轨、多字幕轨。

| 设置项 | 说明 |
|--------|------|
| 视频编码 | H.264 / H.265 / VP9 |
| 音频轨道 | 支持多个音轨 |
| 字幕轨道 | 支持多个字幕轨 |
| 章节 | 支持 DVD/CD 章节 |

## 多格式裁切导出（v1.7.0 新增）

Rust 层 FFmpeg scale+crop filter，无需前端二次转码，直接输出多平台适配尺寸：

| 格式 | 比例 | 输出尺寸 | 推荐场景 |
|------|------|----------|----------|
| **竖屏** | 9:16 | 1080×1920 | 抖音 / 快手 / TikTok / Instagram Reels |
| **方屏** | 1:1 | 1080×1080 | 小红书 / Instagram Feed |
| **横屏** | 16:9 | 1920×1080 | B站 / YouTube / 微博视频 |

FFmpeg filter 实现：

```bash
# 竖屏 9:16（从原始画面中心裁切）
scale=1080:1920:force_original_aspect_ratio=decrease,crop=1080:1920:(iw-1080)/2:(ih-1920)/2

# 方屏 1:1（取 min(iw,ih) 边长裁切中心）
scale=min(iw,ih):min(iw,ih),crop=min(iw,ih):min(iw,ih)
```

::: tip AI 拆条 × 多格式导出
在 **AI 拆条**流程中，选中片段后可以一次性导出多个格式（如同时生成竖屏+方屏版本），实现「一鱼多吃」。
:::

## 专业软件项目格式

### 剪映草稿 (.draft)

导出为剪映可以读取的项目文件。

```
导出后：
1. 打开剪映专业版
2. 点击「导入」→「剪映草稿」
3. 选择 .draft 文件
4. 在剪映中继续精细编辑
```

**支持的内容**：
- ✅ 视频片段和时间轴
- ✅ 字幕轨道
- ✅ 基础转场
- ✅ 文本和贴纸
- ⚠️ 部分特效可能无法完美还原

### Adobe Premiere Pro (.prproj)

导出为 Premiere Pro 项目文件。

```
导出后：
1. 打开 Premiere Pro
2. 点击「文件」→「导入」
3. 选择 .prproj 文件
4. 在 PR 中进行专业后期
```

**支持的内容**：
- ✅ 视频片段和时间轴
- ✅ 字幕轨道
- ✅ 音频轨道
- ✅ 基础调色
- ⚠️ 部分第三方插件效果可能丢失

### DaVinci Resolve (.drp)

导出为 DaVinci Resolve 项目文件。

```
导出后：
1. 打开 DaVinci Resolve
2. 点击「文件」→「导入」
3. 选择 .drp 文件
4. 在 DaVinci 中进行专业调色和剪辑
```

**支持的内容**：
- ✅ 完整的项目结构
- ✅ 视频和音频轨道
- ✅ 字幕轨道
- ✅ 调色信息

### Final Cut Pro (.fcpxml)

导出为 Final Cut Pro 项目文件。

## 平台优化导出

### 抖音 / 快手

| 设置项 | 推荐值 |
|--------|--------|
| 分辨率 | 1080 × 1920（竖屏）|
| 帧率 | 30fps / 60fps |
| 编码 | H.265 |
| 码率 | 8-15 Mbps |
| 音频 | AAC 192kbps |
| 文件大小 | ≤ 2GB（单条）|

### B站

| 设置项 | 推荐值 |
|--------|--------|
| 分辨率 | 1920 × 1080 或 3840 × 2160 |
| 帧率 | 30fps / 60fps |
| 编码 | H.264 |
| 码率 | 1080P: 10-20 Mbps / 4K: 30-50 Mbps |
| 音频 | AAC 320kbps |

### YouTube

| 设置项 | 推荐值 |
|--------|--------|
| 分辨率 | 1920 × 1080 或 3840 × 2160 |
| 帧率 | 24fps / 30fps / 60fps |
| 编码 | H.264 |
| 码率 | 1080P: 8-12 Mbps / 4K: 35-50 Mbps |
| 音频 | AAC 320kbps |
| 色彩空间 | Rec.709 |

### 微信视频号

| 设置项 | 推荐值 |
|--------|--------|
| 分辨率 | 1080 × 1920（竖屏）或 1920 × 1080（横屏）|
| 帧率 | 30fps |
| 编码 | H.264 |
| 码率 | 6-10 Mbps |
| 文件大小 | ≤ 1GB |

### 小红书

| 设置项 | 推荐值 |
|--------|--------|
| 分辨率 | 1080 × 1350（3:4）或 1080 × 1920（9:16）|
| 帧率 | 30fps |
| 编码 | H.264 |
| 码率 | 5-10 Mbps |

## 导出质量预设

CutDeck 内置多个导出预设，方便快速选择：

| 预设名称 | 设置 | 适用场景 |
|----------|------|----------|
| **最高质量** | 4K, H.264, 50Mbps | 存档、母版 |
| **网络发布** | 1080P, H.265, 10Mbps | 抖音/B站/视频号 |
| **快速预览** | 720P, H.264, 5Mbps | 快速预览、传输 |
| **省空间** | 1080P, H.265, 3Mbps | 存储受限场景 |
| **专业后期** | 4K, ProRes 422 | PR/DaVinci 二次编辑 |

## HDR 导出

CutDeck 支持 HDR 视频导出：

| HDR 格式 | 色域 | 比特深度 | 适用场景 |
|----------|------|----------|----------|
| HDR10 | Rec.2020 | 10-bit | YouTube、通用 |
| HDR10+ | Rec.2020 | 10-bit | 三星等设备 |
| Dolby Vision | Rec.2020 | 12-bit | Apple TV、Netflix |
| HLG | Rec.2020 | 10-bit | 广播标准 |

::: warning 注意
HDR 导出需要源素材本身支持 HDR（拍摄时使用 HDR 模式）。
:::

## 批量导出

使用 [批量处理](./batch-processing.md) 功能，可以一次性导出多个视频：

1. 在批量处理面板勾选需要导出的项目
2. 选择 **「批量导出」**
3. 设置统一的导出格式和路径
4. 点击 **「开始导出」**

## 常见问题

### 导出文件太大？

- 🔽 降低码率（从 20Mbps 降到 10Mbps）
- 📐 降低分辨率（从 4K 降到 1080P）
- 🔄 切换 H.265 编码（同等质量文件更小）

### 导出后视频有花屏？

- 💾 检查磁盘空间是否充足
- 🔄 更换输出目录（避免网络磁盘）
- 🔧 尝试更换编码（H.264 替代 H.265）

### 剪映/PR 无法导入？

- 🔍 确认导出格式正确（.draft / .prproj）
- 🔄 重新导出项目文件
- 📝 查看 [故障排除](./troubleshooting.md)
