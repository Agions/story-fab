# 短视频创作指南

本指南教你如何使用 StoryForge 从零创作一条专业级短视频。

---

## 短视频创作流程

```
素材准备 → AI 策划 → 智能剪辑 → 字幕包装 → 导出发布
```

---

## 第一步：素材准备

### 收集素材

| 素材类型 | 说明 |
|----------|------|
| 原始视频 | 拍摄或下载的视频素材 |
| 图片素材 | 封面图、配图、背景图 |
| 音频素材 | 背景音乐、音效 |
| 文字素材 | 标题文案、解说词 |

### 素材要求

::: tip 推荐标准
- 分辨率：≥ 1080P
- 格式：MP4 (H.264/H.265)
- 时长：建议 1-3 分钟（单条短视频）
- 画面：稳定、光线充足
:::

### 导入素材

1. 点击 **「新建项目」**
2. 选择 **「导入素材」** 或直接拖拽文件
3. 等待素材加载完成

---

## 第二步：AI 策划

### 内容分析

点击 **「AI 分析」**，StoryForge 会自动：

- 📊 分析视频叙事结构
- 🎯 识别高光时刻与关键帧
- 🎭 分析情感曲线
- 📝 提取关键内容要点

### 生成文案

StoryForge 可以根据视频内容 **自动生成解说文案**：

1. 选择分析完成的视频
2. 点击 **「生成文案」**
3. 选择文案风格：
   - **专业正式** — 适合知识科普、企业宣传
   - **轻松活泼** — 适合娱乐、生活的分享
   - **情感共鸣** — 适合故事、情感类内容
4. AI 自动生成文案，可在线编辑

### 生成配音

1. 选择已生成的文案
2. 点击 **「生成配音」**
3. 选择音色（见下表）
4. 调整语速和音调
5. 预览并确认

| 音色风格 | 适用场景 |
|----------|----------|
| 男声-磁性 | 故事叙述、知识讲解 |
| 女声-温柔 | 情感、生活、美妆 |
| 男声-活力 | 运动、科技、游戏 |
| 女声-甜美 | 美食、旅行、时尚 |
| 英文-专业 | 国际内容、配音 |

---

## 第三步：智能剪辑

### 选择剪辑模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **自动剪辑** | AI 全自动生成 | 快速产出、素材丰富 |
| **半自动剪辑** | AI 提供建议，你来决策 | 需要人工把控质量 |
| **手动剪辑** | 传统时间线编辑 | 专业创作、精细控制 |

### 自动剪辑参数

| 参数 | 说明 |
|------|------|
| 目标时长 | 设置输出视频的总时长 |
| 节奏感 | 慢 / 中 / 快（影响剪辑点间隔） |
| 转场效果 | 无 / 基础 / 创意 |
| 镜头语言 | 平稳 / 动感 / 电影感 |
| 高光密度 | 低 / 中 / 高（高光片段占比） |

### 剪辑预览

- 🎬 实时预览剪辑效果
- ✏️ 在线调整剪辑点
- 🔄 切换不同的 AI 剪辑方案
- 📝 单独替换某个片段

---

## 第四步：字幕包装

### 字幕生成

StoryForge 支持 **多种字幕生成方式**：

| 方式 | 说明 |
|------|------|
| ASR 自动生成 | 从音频自动识别文字（需 AI 服务） |
| 导入字幕 | 导入已有的 SRT/ASS 字幕文件 |
| 手动输入 | 手动输入字幕内容 |

### 字幕样式

StoryForge 提供丰富的字幕样式模板：

| 样式类型 | 特点 |
|----------|------|
| 电影字幕 | 黑底白字，居中显示 |
| 弹出字幕 | 跟随说话人位置 |
| 动感字幕 | 带动画效果（打字机、缩放等） |
| 封面字幕 | 用于标题、关键信息展示 |

### 自定义样式

1. 选择字幕轨道
2. 点击 **「样式设置」**
3. 自定义：
   - 字体、大小、颜色
   - 背景样式（描边、阴影、背景色）
   - 位置（居中、左侧、跟随人脸）
   - 动画效果

### 封面生成

AI 自动生成视频封面：

1. 从视频中智能选取精彩帧
2. AI 生成多个封面方案
3. 选择最喜欢的一个
4. 可手动上传自定义封面

---

## 第五步：导出发布

### 选择导出格式

| 格式 | 说明 | 推荐场景 |
|------|------|----------|
| **MP4** | 通用格式，画质高 | 各平台直接发布 |
| **MOV** | Apple 生态，高码率 | 进一步编辑 |
| **剪映草稿** | 导入剪映继续编辑 | 精细调色、特效 |
| **PR 项目** | 导入 Premiere Pro | 专业后期 |

### 平台优化导出

| 平台 | 推荐设置 |
|------|----------|
| 抖音/快手 | 9:16 竖屏，1080P，H.265 |
| B站 | 16:9 横屏，1080P/4K，H.264 |
| YouTube | 16:9 横屏，4K，H.264 |
| 微信视频号 | 16:9 或 9:16，1080P |
| 小红书 | 3:4 或 9:16，1080P |

### 发布设置

导出完成后，可以直接在 StoryForge 中：

- 📋 复制视频文件到指定目录
- 📤 一键上传到指定平台（需配置平台账号）
- 🖼️ 生成配套的封面图和文案

## 创作案例

### 案例一：知识科普视频

```
1. 导入素材 → 讲座/教程视频
2. AI 分析 → 识别关键知识点
3. 生成文案 → 专业正式风格
4. 自动剪辑 → 去除冗余内容
5. 添加字幕 → 动感字幕样式
6. 导出发布 → B站/抖音
```

### 案例二：生活 vlog

```
1. 导入素材 → 多个 vlog 片段
2. AI 分析 → 识别精彩瞬间
3. 选择高光 → 人工挑选片段
4. 套用模板 → 「旅行 vlog」模板
5. 添加音乐 → 匹配节奏
6. 导出发布 → 各平台
```

---

## 技巧与最佳实践

| 技巧 | 说明 |
|------|------|
| 🎯 精选素材 | 素材质量决定成品质量，拍摄时多拍精彩片段 |
| 📝 提前规划 | 写好文案框架，AI 生成的文案更精准 |
| 🎵 重视音乐 | 背景音乐是短视频的灵魂，选择节奏匹配的内容 |
| ⏱️ 控制时长 | 黄金时长：抖音 15-30s，B站 3-5 分钟 |
| 📱 竖屏优先 | 如果主要发抖音/快手，直接用竖屏素材 |
