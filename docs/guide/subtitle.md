# 字幕处理详解

StoryForge 提供强大的字幕处理能力，支持 **自动识别、手动编辑、样式定制和多格式导出**。

---

## 字幕生成方式

### 1. ASR 自动生成

ASR（自动语音识别）从视频音频中自动提取文字。

```
操作流程：
1. 选择视频轨道
2. 点击「字幕」→「自动生成」
3. 选择识别语言（中文/英文/日文/韩文等）
4. 等待识别完成
```

| 语言 | 识别准确率 | 适用场景 |
|------|-----------|----------|
| 中文普通话 | ~95% | 国内视频 |
| 英文 | ~97% | 英文原声 |
| 日语 | ~90% | 日漫、日综 |
| 粤语 | ~93% | 广东地区 |

::: tip 准确率提升技巧
- 音频清晰、无背景噪音
- 说话人发音标准、无严重口音
- 背景音乐音量适中
:::

### 2. 导入字幕文件

支持导入已有字幕文件：

| 格式 | 扩展名 | 支持情况 |
|------|--------|----------|
| SRT | `.srt` | ✅ 完全支持 |
| ASS/SSA | `.ass` / `.ssa` | ✅ 完全支持（含样式） |
| VTT | `.vtt` | ✅ 完全支持 |
| LRC | `.lrc` | ✅ 支持（歌词格式） |
| TXT | `.txt` | ✅ 支持（按行分割） |

```
导入操作：
1. 点击「字幕」→「导入字幕」
2. 选择字幕文件
3. 系统自动识别时间轴（无时间轴文件按行均分时长）
4. 确认导入
```

### 3. 手动输入

```
操作流程：
1. 在字幕轨道上点击「添加字幕」
2. 输入文字内容
3. 拖拽调整字幕时间轴
4. 设置字幕样式
```

## 字幕编辑

### 时间轴编辑

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 拖拽字幕 | 鼠标拖拽 | 移动字幕位置 |
| 调整时长 | 拖拽边缘 | 改变字幕显示时长 |
| 拆分字幕 | `S` | 将一个字幕拆为两个 |
| 合并字幕 | `M` | 将两个相邻字幕合并 |
| 批量调整 | `Shift + 拖拽` | 批量移动多个字幕 |

### 文本编辑

- ✏️ 直接双击字幕编辑文字
- 🔤 支持多语言混合输入
- 📝 支持特殊符号和 emoji

### AI 字幕修正

对 ASR 识别的字幕进行 AI 智能修正：

1. 选择需要修正的字幕
2. 点击「AI 修正」
3. AI 自动修正错别字、标点
4. 确认修正结果

::: tip 适用场景
- 人名、地名、专业术语修正
- 添加正确标点
- 口语化表达规范化
:::

## 字幕样式

### 内置样式模板

| 样式名称 | 特点 | 适用场景 |
|----------|------|----------|
| 电影字幕 | 黑底白字，居中 | 电影、剧集 |
| 弹出字幕 | 半透明背景，跟随位置 | 访谈、对话 |
| 动感字幕 | 打字机/缩放动画 | 短视频、社交媒体 |
| 简约白字 | 白色无背景 | 知识讲解 |
| 毛玻璃字幕 | 毛玻璃背景效果 | 时尚、生活类 |

### 自定义样式

点击 **「样式设置」** 进入自定义面板：

#### 字体设置

| 参数 | 选项 |
|------|------|
| 字体 | 系统字体 / 自定义字体 |
| 字号 | 16px - 72px |
| 粗细 | 常规 / 中等 / 加粗 |
| 颜色 | 任意颜色 |
| 透明度 | 0% - 100% |

#### 背景设置

| 参数 | 选项 |
|------|------|
| 背景颜色 | 任意颜色 |
| 透明度 | 0% - 100% |
| 圆角 | 0px - 20px |
| 内边距 | 0px - 20px |
| 描边 | 颜色 + 宽度 |
| 阴影 | 颜色 + 模糊度 + 偏移 |

#### 位置设置

| 参数 | 选项 |
|------|------|
| 水平位置 | 居中 / 左侧 / 右侧 / 自定义 |
| 垂直位置 | 底部 / 居中 / 顶部 |
| 偏移量 | 自定义 X/Y 偏移 |

#### 动画效果

| 动画类型 | 说明 |
|----------|------|
| 无动画 | 直接显示 |
| 淡入淡出 | 透明度渐变 |
| 打字机 | 逐字显示 |
| 缩放 | 从小到大出现 |
| 上滑 | 从下往上滑入 |

## 翻译字幕

### AI 翻译

使用 AI 将字幕翻译为其他语言：

```
操作流程：
1. 选择字幕轨道
2. 点击「字幕」→「翻译」
3. 选择目标语言
4. 选择翻译质量（快速/标准/高质量）
5. AI 生成翻译字幕
```

### 翻译字幕应用

| 应用方式 | 说明 |
|----------|------|
| **替换原字幕** | 用翻译字幕替换原文字幕 |
| **双语字幕** | 原文字幕 + 翻译字幕同时显示 |
| **上下排列** | 原文字幕在下方，翻译字幕在上方 |

::: warning 注意
AI 翻译会消耗翻译 API 配额，推荐使用 DeepSeek 或 Qwen，性价比最高。
:::

## 多轨字幕

StoryForge 支持 **多轨字幕**，同时显示多条字幕：

| 轨道 | 用途 |
|------|------|
| 字幕轨 1 | 主要解说字幕 |
| 字幕轨 2 | 对话翻译 / 双语字幕 |
| 字幕轨 3 | 补充说明 / 注释 |

## 字幕导出

### 导出格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| SRT | `.srt` | 通用字幕格式 |
| ASS | `.ass` | 支持高级样式和动画 |
| VTT | `.vtt` | Web 标准字幕格式 |
| 封包视频 | `.mp4` / `.mov` | 硬字幕直接烧录进视频 |

### 导出设置

| 选项 | 说明 |
|------|------|
| 编码 | UTF-8 / UTF-16 |
| 音频同步 | 自动校准时间轴 |
| 样式保留 | 是否保留 ASS 样式信息 |

::: tip 建议
如果要二次编辑，推荐导出 **ASS 格式**，保留所有样式和动画信息。
如果要直接发布，推荐选择 **封包视频**，硬字幕烧录进去。
:::

## 常见问题

### 字幕不同步怎么办？

1. 点击 **「校准」** 按钮
2. 播放视频，找到同步点
3. 点击「标记」，AI 自动调整全部字幕时间轴

### ASR 识别有噪音？

- 🎤 建议先对音频进行降噪处理
- 🎯 开启 **「人声增强」** 选项
- 📝 识别后用 AI 修正错字

### 如何批量处理字幕？

使用 [批量处理](./batch-processing.md) 功能，批量导入、生成、翻译字幕。
