# AI 工作流

> StoryForge 的核心 AI 处理流程

## 工作流概览

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   视频上传   │ -> │   AI 分析   │ -> │   智能剪辑   │
└─────────────┘    └─────────────┘    └─────────────┘
       │                  │                  │
       v                  v                  v
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  素材管理    │    │  内容理解   │    │  脚本生成   │
└─────────────┘    └─────────────┘    └─────────────┘
                                            │
                                            v
                                    ┌─────────────┐
                                    │   配音合成   │
                                    └─────────────┘
                                            │
                                            v
                                    ┌─────────────┐
                                    │   视频导出   │
                                    └─────────────┘
```

## 详细流程

### 1. 视频上传

**支持格式**：MP4, MOV, WebM, AVI
**最大文件**：2GB
**处理方式**：本地处理，不上传服务器

### 2. AI 分析

| 分析项 | 说明 |
|--------|------|
| 场景检测 | 识别视频中的场景切换点 |
| 物体识别 | 识别视频中的主体对象 |
| 音频分析 | 提取背景音乐和人声 |
| 字幕识别 | OCR 提取画面文字 |
| ASR | 语音转文字 |

### 3. 智能剪辑

**剪辑模式**：

| 模式 | 说明 |
|------|------|
| AI 自动剪辑 | AI 自动识别精彩片段 |
| 自定义剪辑 | 手动选择片段 |
| 混剪模式 | 多素材智能混剪 |

### 4. 脚本生成

**生成选项**：

| 选项 | 说明 |
|------|------|
| 文案风格 | 正式/轻松/幽默/情感 |
| 目标受众 | 大众/专业/学生等 |
| 视频长度 | 短视频/中视频/长视频 |
| 语气选择 | 友好/权威/热情/平静 |

### 5. 配音合成

**配音选项**：

| 选项 | 说明 |
|------|------|
| 声音选择 | 女声/男声/中性 |
| 语速调节 | 0.5x - 2.0x |
| 音调调节 | 低/中/高 |
| 音量调节 | 0% - 200% |

### 6. 视频导出

**输出格式**：MP4, MOV, WebM
**分辨率**：720p, 1080p, 4K
**帧率**：24, 30, 60 FPS

## 工作流配置

### AI 模型选择

可以在设置中配置默认 AI 模型，不同任务可选择不同模型：

| 任务 | 推荐模型 |
|------|----------|
| 脚本生成 | GPT-5.3 / Claude 4.6 |
| 内容分析 | Gemini 3 Ultra |
| 中文处理 | Qwen 3.5 / Kimi k2.5 |
| 成本控制 | DeepSeek R1 |

### 批处理配置

支持批量处理多个视频：

1. 添加多个视频到批处理队列
2. 统一设置处理参数
3. 启动批量处理
4. 查看处理进度和结果

### 自动化脚本

支持通过脚本自动执行工作流：

```bash
# 导入项目
storyforge import --video input.mp4

# 执行 AI 分析
storyforge analyze --project <id>

# 生成剪辑
storyforge clip --project <id> --mode auto

# 导出视频
storyforge export --project <id> --format mp4 --quality 1080p
```

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| AI 分析失败 | 检查 API 密钥配置 |
| 处理速度慢 | 降低输出分辨率 |
| 导出失败 | 检查磁盘空间 |
| 音画不同步 | 重新生成配音 |
