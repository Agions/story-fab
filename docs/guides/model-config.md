# 模型配置

## 配置原则

- 模型选项应来自 API 密钥管理中已配置的模型
- UI 下拉列表与后端可用模型保持一致
- 不展示已废弃模型，避免误选

## 推荐模型映射（2026-03）

- OpenAI：`gpt-5.3-codex`
- Anthropic：`claude-sonnet-4-6`
- Google：保留 2 个主模型选项（按项目策略）
- 智谱：`glm-5`
- 月之暗面：`kimi-k2.5`

## 配置检查清单

1. API Key 存在且启用
2. 模型名与供应商匹配
3. 选择模型后可成功发起请求
4. 项目保存后可回显上次模型

## 常见错误

- 下拉有旧模型：检查模型常量是否已清理
- 模型不可用：确认 key 与 endpoint
- 文案显示异常：检查 i18n key 是否仍引用旧字段
