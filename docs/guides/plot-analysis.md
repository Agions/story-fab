# 剧情分析模式 (Plot Analysis Mode)

> StoryForge v1.1.0 新增功能

## 概述

剧情分析模式是 StoryForge 全新引入的**叙事优先**剪辑方法。与传统基于节奏或情感的剪辑不同，它专注于理解和保留视频的**故事结构**。

## 核心概念

### 剧情图谱 (Plot Timeline)

AI 自动分析视频内容后，会生成一个可视化的"剧情图谱"，包含：

| 节点类型 | 说明 | 视觉标记 |
|---------|------|---------|
| 背景铺垫 (Setup) | 建立世界观和角色 | 🏠 |
| 上升情节 (Rising Action) | 矛盾冲突升级 | 📈 |
| 高潮 (Climax) | 故事最精彩时刻 | 🔥 |
| 情感转折 (Emotional Beat) | 情绪变化点 | 💫 |
| 对话场景 (Dialogue) | 角色对话 | 💬 |
| 动作场景 (Action) | 激烈动作 | ⚡ |
| 转场 (Transition) | 场景切换 | ➡️ |

### 情感轨迹

AI 会追踪视频中的情感变化，生成情感曲线图：

```
兴奋 ┤    ╭─╮        ╭──╮
     │   ╭╯  ╰─╮   ╭╯    ╰╮
平静 ┤──╯       ╰──╯        ╰──
     └────────────────────────→ 时间
```

## 使用流程

### 1. 上传视频

拖拽或选择视频文件到 StoryForge。

### 2. 选择剧情分析模式

在剪辑模式选择器中，选择：

```
🎭 剧情分析模式
  └ 基于视频内容理解，自动分析剧情结构并剪辑
```

### 3. AI 分析

等待 AI 完成以下分析：

- ✅ 场景边界检测
- ✅ 音频转写 (ASR)
- ✅ 情感分析
- ✅ 剧情结构理解

### 4. 查看剧情图谱

分析完成后，您将看到：

- **时间线视图**：显示所有剧情节点
- **节点详情**：每个节点的时间戳、类型、重要性
- **情感曲线**：可视化情感变化

### 5. 选择要保留的内容

您可以选择：

| 标签 | 说明 |
|-----|------|
| 高光片段 | 重要性 ≥ 7 的节点 |
| 剧情高潮 | Climax 类型节点 |
| 情感转折 | Emotional Beat 类型节点 |
| 全部保留 | 完整叙事 |

### 6. 生成剪辑

AI 根据您的选择，自动生成剪辑时间线。

### 7. 预览和导出

预览最终效果，进行微调，然后导出。

## 输出版本

剧情分析模式支持三种输出版本：

### 📼 剧情完整版 (Full Narrative)

保留完整的故事结构，适合：
- 纪录片
- 采访整理
- 完整活动记录

### ✂️ 精华版 (Highlights Reel)

只保留高光时刻，适合：
- 社交媒体分享
- 预告片
- 精选集锦

### ⚡ 高能混剪版 (Intense Mix)

只保留高潮和动作场景，适合：
- 快节奏混剪
- 精彩片段合集
- 视觉冲击展示

## 技术实现

### 处理流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  视频输入   │ -> │  场景检测   │ -> │  ASR 转写   │
└─────────────┘    └─────────────┘    └─────────────┘
                                            │
                                            ▼
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  剪辑输出   │ <- │  节点生成   │ <- │  LLM 理解   │
└─────────────┘    └─────────────┘    └─────────────┘
```

### API 调用

```typescript
import { plotAnalysisService } from '@/core/services/plotAnalysis.service';

// 分析视频剧情
const timeline = await plotAnalysisService.analyzePlot(videoInfo, {
  detectCharacters: true,
  analyzeNarrativeArc: true,
});

// 生成特定版本剪辑
const clips = plotAnalysisService.generateEditTimeline(timeline, 'highlights');
```

## 适用场景

### ✅ 适合

- 有明确叙事结构的视频（故事、纪录片、活动记录）
- 需要保留情感节奏的长视频
- 希望保持叙事连贯性的混剪

### ❌ 不适合

- 纯音乐/舞蹈视频（无对话/叙事）
- 快节奏切换的混剪（需要传统 Smart Clip）
- 完全随机的内容

## 常见问题

**Q: 为什么分析需要较长时间？**

A: 剧情分析涉及多轮 AI 处理，包括场景检测、ASR 转写、情感分析和 LLM 剧情理解。视频越长，分析时间越长。

**Q: 可以同时使用剧情分析和其他剪辑模式吗？**

A: 可以。您可以在剧情分析后，叠加使用 Smart Clip 进行进一步优化。

**Q: 如何提高剧情分析的准确性？**

A: 提供更高质量的音频（清晰的人声）可以提高 ASR 准确率，从而提升整体分析质量。
