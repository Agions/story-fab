项目结构

StoryForge 使用 React + TypeScript + Vite 构建。

目录结构

StoryForge/
├── src/
│   ├── components/     # React 组件
│   │   ├── AIClipPanel/ # AI 剪辑面板
│   │   ├── AIPanel/     # AI 功能面板
│   │   ├── Dashboard/   # 仪表板
│   │   └── editor/      # 编辑器组件
│   ├── core/           # 核心逻辑
│   │   ├── services/    # 服务层
│   │   ├── types/       # TypeScript 类型
│   │   └── constants/   # 常量定义
│   ├── pages/          # 页面组件
│   ├── hooks/          # 自定义 Hooks
│   └── utils/          # 工具函数
├── public/             # 静态资源
├── package.json
└── vite.config.ts     # Vite 配置

主要模块

  • components/ - UI 组件库
  • core/services/ - AI 服务、视频处理等核心服务
  • core/types/ - TypeScript 类型定义
  • pages/ - 页面级组件
  • hooks/ - 可复用的 React Hooks
