# FAQ

## 安装相关

### 应用闪退 / 黑屏怎么办？

1. 确认显卡驱动最新
2. 尝试带参数启动：`StoryFab.exe --disable-gpu`（Windows）或 `./StoryFab --disable-gpu`（macOS/Linux）
3. 删除配置目录后重启：`~/.story-fab/`（会清除项目缓存，谨慎操作）

### 首次启动后模型下载失败？

网络受限时，可手动下载后放入模型目录：

1. 访问 [HuggingFace](https://huggingface.co/ggerganov/whisper.cpp) 下载 `ggml-base.bin`
2. 放入 `~/.story-fab/models/whisper/`（目录不存在则新建）
3. 重启应用

### 支持哪些语言？

- Whisper 转录：支持 99 种语言，含中 / 英 / 日 / 韩 / 法等
- TTS 配音：Edge TTS 支持 40+ 中文音色；Azure TTS 支持 100+ 音色
- 界面语言：中文（简体）/ English

## 转录相关

### 转录速度太慢？

| 模型 | 1小时音频耗时 | 建议 |
| --- | --- | --- |
| `tiny` | 3-5 分钟 | 实时 / 快速预览 |
| `base` | 5-8 分钟 | 日常使用 ⚡ |
| `small` | 15-25 分钟 | 高质量转录 |
| `medium` | 30-60 分钟 | 专业用途 |

在「设置 → 高级」中可调低模型等级。转录速度受 CPU 和内存影响大。

### 转录结果不准确（中文）？

1. 尝试 `small` 模型，中文准确率提升显著
2. 视频噪音大时，可先运行「音频降噪」转录效果更好
3. Whisper 对方言识别偏弱，建议视频内容为标准普通话

### 歌词 / 背景音乐导致字幕有误？

这是 Whisper 的已知局限。两种处理方式：

1. 先运行「降噪」过滤背景音乐再转录
2. 直接在转录完成后编辑错误文本

## 解说模式相关

### 脚本风格不满意？

在「高级」选项卡中注入 Prompt 覆盖默认行为：

```
"这是一个纪录片风格的解说，避免使用网络用语，
语速中等偏慢，重点突出历史背景。"
```

支持的风格：纪录片、综艺、教学、影评、Vlog。

### 生成的脚本太长或太短？

- 可在「审核」步骤手动裁剪 / 扩充
- 在 Prompt 中明确指定字数："生成 500 字左右的解说"
- 调整视频片段长度，短片段自动匹配少文字

### TTS 配音听起来像机器人？

- Edge TTS 默认音色差异大，多试几个，推荐 `zh-CN-XiaoxiaoNeural`
- Azure TTS 自然度更高，推荐使用
- 支持导入自己的录音文件替换任何段落

## 导出相关

### 导出失败怎么办？

| 错误 | 原因 | 解决方案 |
| --- | --- | --- |
| `磁盘空间不足` | 输出目录空间不够 | 换输出目录 |
| `编码器错误` | H.265 显卡不支持 | 切回 H.264 |
| `音频编码失败` | 采样率不匹配 | 确认视频采样率为 44100 / 48000 Hz |
| `写入权限不足` | 输出目录无写入权限 | 修改目录权限或换目录 |

### 导出的视频在某些播放器无声音？

- 导出时确认音频编码为 `AAC`（MP3 在老播放器可能不兼容）
- 采样率选择 `44100 Hz` 或 `48000 Hz`

### 字幕样式在手机上显示太小？

导出时选择「字幕样式 C — 大号描边」，或在自定义中把字号设为屏幕高度的 6-8%。

## 性能相关

### 预览卡顿怎么办？

- 关闭「实时预览」→ 改为「按需预览」（点击才播放）
- 降低预览分辨率：在「设置 → 性能」中选择「标清预览」
- 将项目文件放在 SSD 上（HDD 处理 1080p/4K 会卡顿）

### 内存不足警告？

- 其他占用内存大的应用保存并退出
- 在「设置 → 性能」中调低 `max_concurrent_tasks`
- 4K 处理建议内存 ≥ 32 GB

### GPU 加速无效？

StoryFab 的 GPU 加速依赖：

- **Windows**：NVIDIA / AMD 显卡驱动最新，支持 NVENC / AMF
- **macOS**：自动使用 VideoToolbox，无需手动配置
- **Linux**：需安装 `vainfo` 支持的驱动

在「设置 → 高级 → 诊断」中可查看 GPU 加速状态。

## 隐私相关

### StoryFab 会上传任何数据吗？

默认情况下 **零上传**：
- 高光检测：纯本地
- Whisper 转录：纯本地推理（模型下载后完全离线）
- Agent Pipeline：本地推理（默认不调用任何 API）

调用 API 服务（如 Azure TTS）时，音频/文本数据发送至对应服务。所有网络调用都可在设置中关闭或切换本地模式。

### 如何完全删除我的项目数据？

1. 在「设置 → 存储」中查找项目路径
2. 手动删除项目目录
3. 清空 `~/.story-fab/cache/transcription/` 中的转录缓存
4. 清空 `~/.story-fab/cache/tts-cache/` 中的 TTS 缓存

所有数据都在本地，不上传任何云端。删除即彻底清除。
