# 工作流

StoryForge 提供完整的 AI 视频创作工作流。

## 创作模式

### 1. 剧情分析模式 🎬

适用于：电影解说、Vlog 整理、纪录片剪辑

**流程：**
1. 上传视频 → 2. AI 分析剧情 → 3. 查看叙事结构 → 4. 生成剪辑建议 → 5. 导出

### 2. AI 解说模式 🎙️

适用于：科普视频、产品介绍、知识分享

**流程：**
1. 上传视频 → 2. AI 分析场景 → 3. 生成解说文案 → 4. 配音合成 → 5. 字幕匹配 → 6. 导出

### 3. 智能混剪模式 🎵

适用于：短视频混剪、音乐 MV、精彩集锦

**流程：**
1. 导入多素材 → 2. BPM 检测 → 3. 节奏分析 → 4. 自动卡点 → 5. 转场合 成 → 6. 导出

### 4. 情感独白模式 🎭

适用于：Vlog 升华、情感纪录片

**流程：**
1. 上传视频 → 2. 情感分析 → 3. 独白生成 → 4. 配音合成 → 5. 电影字幕 → 6. 导出

## 导出格式

| 格式 | 适用场景 |
|------|----------|
| 剪映草稿 | 导入剪映继续编辑 |
| Premiere XML | Adobe Premiere Pro |
| FCPXML | Final Cut Pro X |
| DaVinci EDL | DaVinci Resolve |
| MP4 | 直接发布 |

## 快捷键

| 功能 | 快捷键 |
|------|--------|
| 播放/暂停 | `Space` |
| 上一帧 | `←` |
| 下一帧 | `→` |
| 剪切 | `Cmd/Ctrl + B` |
| 撤销 | `Cmd/Ctrl + Z` |
| 重做 | `Cmd/Ctrl + Shift + Z` |
