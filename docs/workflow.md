## Step 1 — 上传素材

### 支持的上传方式

| 方式 | 操作 |
|------|------|
| 拖拽上传 | 直接拖入应用窗口 |
| 点击上传 | 点击按钮选择文件 |
| 批量上传 | 一次选择多个文件 |

### 文件要求

| 项目 | 要求 |
|------|------|
| 支持格式 | MP4 / MOV / WebM / AVI / MKV |
| 单文件大小 | 最大 2GB |
| 批量数量 | 最多 10 个同时上传 |

---

## Step 2 — AI 分析

上传完成后，系统自动进行全量分析：

### 分析项目

| 分析类型 | 说明 | 耗时参考 |
|----------|------|---------|
| 场景检测 | 识别镜头边界，切分场景 | ~视频时长×0.5s |
| 音频峰值 | 标记笑声/掌声/音乐高潮 | ~视频时长×0.3s |
| 情感分析 | 画面色调与情绪变化追踪 | ~视频时长×1s |
| ASR 转写 | 语音转文字（含时间戳）| ~视频时长×0.8s |
| OCR 提取 | 画面中可见文字识别 | ~视频时长×1.5s |

> 💡 分析完成后会自动生成**剧情图谱**（剧情分析模式）或**高光标记**（智能混剪模式）。

---

## Step 3 — 选择剪辑模式

根据内容类型选择合适的工作模式：

### 🎭 剧情分析模式

```
剧情图谱 → 选择叙事元素 → 生成剪辑方案 → 预览微调
```

适合：纪录片、访谈、剧情类视频
输出：完整叙事版本 / 精华版 / 高能混剪版

### 🔥 智能混剪模式

```
高光标记 → 选择精彩片段 → 自动拼接 → 添加转场
```

适合：活动回顾、体育高光、综艺片段
输出：节奏感强的精华集锦

### 🎤 AI 解说模式

```
分析内容 → 生成解说词 → 选择音色 → 合成配音 → 导出成片
```

适合：知识类视频、产品介绍、纪录片旁白
输出：配音完整版视频

### 📝 第一人称视角模式

```
分析素材 → 重构叙事角度 → 自动配音 → 导出
```

适合：Vlog、旅拍、个人故事类内容
输出：具有个人叙事感的视频

---

## Step 4 — 预览与微调

### 可调整项目

| 调整项 | 说明 |
|--------|------|
| 片段顺序 | 拖拽调整片段排列 |
| 片段裁剪 | 精细调整入出点 |
| 转场效果 | 淡入淡出 / 滑动 / 溶解 等 |
| 字幕样式 | 字体 / 颜色 / 位置 / 大小 |
| 音量平衡 | 背景音乐 / 原声 / 配音比例 |

### 预览操作

- **播放控制** — 空格键播放/暂停
- **时间跳转** — 点击时间线跳转
- **缩放** — 滚轮缩放时间线
- **撤销** — Ctrl+Z 撤销上一步
- **重做** — Ctrl+Shift+Z 重做

---

## Step 5 — 导出发布

### 配置导出参数

| 参数 | 可选值 |
|------|--------|
| 格式 | MP4 / WebM / MOV / MKV |
| 分辨率 | 720p / 1080p / 4K |
| 质量 | 低 / 中 / 高 / 超高 |
| 帧率 | 24 / 25 / 30 / 60 fps |

### 平台预设

| 平台 | 推荐设置 |
|------|---------|
| 抖音 / 快手 | 1080p / MP4 / 高质量 / 9:16 或 16:9 |
| B站 | 1080p / MP4 / 高质量 / 16:9 |
| YouTube | 4K / MP4 / 超高质量 / 16:9 |
| 微信视频号 | 1080p / MP4 / 中等质量 / 16:9 或 9:16 |
| 小红书 | 1080p / MP4 / 高质量 / 9:16 |

## 完整流程示例

以「剧情分析模式」剪辑一部纪录片为例：

```
1. 上传纪录片视频（45分钟）
       │
       ▼
2. AI 完成全量分析
   - 识别 128 个场景
   - 转写 38 段对话
   - 追踪情感曲线
       │
       ▼
3. 查看剧情图谱
   - 开场铺垫 (0:00-3:00)
   - 背景交代 (3:00-8:00)
   - 冲突建立 (8:00-15:00)
   - 高潮 (15:00-18:00)
   - 结局 (18:00-22:00)
       │
       ▼
4. 选择剪辑策略
   - 保留所有关键对话
   - 压缩背景铺垫段落
   - 精保留高潮段落
       │
       ▼
5. 生成 3 个版本
   - 📼 剧情完整版（22分钟）
   - ✂️ 精华版（8分钟）
   - ⚡ 高能混剪版（5分钟）
       │
       ▼
6. 预览微调 → 导出
```
